# MyBestWebsite

Bu küçük proje, `sites.json` dosyasından veri çekerek kategorilere ayrılmış favori sitelerinizi gösterir.

Çalıştırma (basit HTTP sunucusu kullanın):

Windows / Python 3 terminalinde:

```powershell
python -m http.server 8000
```

Tarayıcıda açın: http://localhost:8000

`sites.json` içinde kategorileri ve site girdilerini düzenleyerek kendi listenizi oluşturabilirsiniz. Görseller yerel dosya isimleri (ör. `ezgifWallpaper.png`) ya da uzak URL olabilir.
